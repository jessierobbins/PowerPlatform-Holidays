# Holidays app for Power Platform
Use Power Platform to add organizational holidays to user calendars.  This repo contains all versions of this solution.  

## HolidaysStandardLicense
[![Holidays Standard License app in Play Mode](/screenshots/HolidaysStandardLicense-PlayMode.jpg)](/screenshots/HolidaysStandardLicense-PlayMode.jpg)
This is the unmanaged version of a standard license solution that requires no Premium license for use and is fully self-contained within a canvas app.  

Upon loading to your environment you will need to edit the Canvas app and update the Formulas reference to your organization's holidays.  The US Federal holidays are loaded in as an example.  
![Holidays Standard License app - Formulas object example](/screenshots/HolidaysStandardLicense-HolidayFormula.jpg)
